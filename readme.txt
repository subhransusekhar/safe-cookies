Tries to create a more secure cookie with random vatiable to avoid reuse of cookies

== Description ==

When you login to wordpress site it saves the auth info in cookies which have a expire time of 2 days. If some body get access of your cookie he can get into your wordpress iste. Use this plugin to increase the security by adding some random variable in the cookie for validation.

== Installation ==

To install the plugin follow these steps :

1. Download the safe-cookies.zip file to your local machine.
2. Unzip the file 
3. Upload "safe-cookies" folder to the "/wp-content/plugins/" directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. You will be prompted to log in again. Do so. This is necessary to set the new cookie.
